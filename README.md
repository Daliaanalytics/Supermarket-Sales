# Supermarket-Sales
## Overview
Analysis of supermarket sales data to identify trends and business insights.

## Tools Used
- Python (Pandas, Matplotlib)
- Power BI

## Key Findings
- Cairo is the top city with 5,480 in total sales
- Electronics is the best category with 8,000 in sales
- April was the best month
- Branch C has the highest customer rating (8.12)

- Supermarket_Sales_Dashboard.pbix — Power BI Dashboard
